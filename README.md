# X Infinite Scroll Stopper

A lightweight browser extension that allows you to stop infinite scrolling
on X (Twitter) with a single click.

This extension gives you control over when new content loads,
helping reduce distraction and endless scrolling.

---

## Features

- One-click enable / disable
- Clear status indicator
- Optional automatic activation on browser startup
- Reload button for open X tabs
- Minimal and lightweight



---

## About the "scan" / loading issue

Sometimes X may continue showing already cached content
even after blocking is enabled.

This is **not a bug**.

If you notice new posts still appearing:
1. Enable blocking
2. Press **Reload X tab** from the extension
3. Scrolling will stop normally

## Installation

### Chrome / Chromium / Brave (Desktop)

1. Download or clone this repository
2. Open `chrome://extensions`
3. Enable **Developer mode**
4. Click **Load unpacked/zip/crx**
5. Select the extension folder

---

### Kiwi Browser (Android)

1. Download the extension folder (or extract the zip)
2. Open `chrome://extensions` in Kiwi
3. Enable **Developer mode**
4. Tap **Load unpacked/zip/crx**
5. Select the folder
